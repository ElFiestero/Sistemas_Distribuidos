# Sistemas_Distribuidos

1.¿Qué ventajas encuentras en trabajar directamente en la nube con GitHub?

Trabajar directamente en la nube con GitHub permite acceder al proyecto desde cualquier lugar, facilita la colaboración en tiempo real, el control de versiones automático y evita problemas de configuración en equipos locales.

2. ¿Qué desafíos tuviste al crear el sitio sin usar herramientas locales?
Uno de los principales desafíos fue la dependencia de la conexión a internet, la limitación en herramientas de desarrollo avanzadas y la dificultad para depurar errores sin un entorno local más completo.

3. ¿Cómo podrías mejorar este sitio web en el futuro?
Podría mejorar el sitio implementando mejor diseño responsivo, optimización de rendimiento, integración de bases de datos y agregar funcionalidades dinámicas con frameworks modernos.

4. ¿Qué aprendiste sobre Sistemas Distribuidos al crear este sitio?
Aprendí que los Sistemas Distribuidos permiten trabajar de forma colaborativa y descentralizada, gestionando recursos y versiones en múltiples nodos, como ocurre al usar repositorios remotos y servicios en la nube.
